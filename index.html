Functions

Functions are nice little blocks of code that can be reused without retyping everything. They are methods of manipulating values. Functions are often used to perform a calculation and return the result, or simply calculating something and changing the value automatically.

For example, here is some quick code that would return a result:


    def area(length):
        return length * length
    
    print('The area of a 2x2 square is ' + area(2))
    # Result: "The area of a 2x2 square is 4"

Next, some code that alters an already existing value:


    changeme = "value 1"
    print(changeme)
    # Result: "value 1"

    def new_message(msg):
        changeme = msg

    print(changeme)
    # Result: "value 1"

    new_message("hello!")
    print(changeme)
    # Result: "hello!"

As you can see in the last example, the code inside of a function only happens when that function is called.

KEEP IN MIND! When you call a function, it runs! That seems self explanatory, but if you're not careful you could run into a situation where it becomes an issue.

Take this situation for example: You have a function that generates a random message. You want to use that message in multiple places, so you try the following:


    import random
    def rand_msg():
        msgs = ["hello", "goodbye", "have a nice day"]
        choice = random.randint(0,len(msgs)-1) # Takes a random number between 0 and 2, the range of indexes of the msgs list
        return msgs[choice]
    
    print('Msg 1: ' + rand_msg())
    print('The same msg: '+ rand_msg())

If you ran this code multiple times, you would notice that the two print statements wouldn't always return the same message. That is because each time you use rand_msg(), the function runs again and a new random message is returned.

To get around this, you can set function results as variables. That feels like a mouthful, but I promise it's not complicated. Editing the code from above, we get:


    import random
    def rand_msg():
        msgs = ["hello", "goodbye", "have a nice day"]
        choice = random.randint(0,len(msgs)-1) # Takes a random number between 0 and 2, the range of indexes of the msgs list
        return msgs[choice]
    
    message = rand_msg()
    print('Msg 1: ' + message)
    print('The same msg: '+ message)

When you use a variable, the function only runs once: when you first assign the variable.

Lists

Lists are used to store multiple items in one variable. Lists are the simplest form of a data collection, and the easiest to work with. Lists are extremely useful in all coding languages.

To define a list, you place values into square brackets:


    a_list = ["this", "is", "a", "list"]
    print(a_list)
    # Result: ["this", "is", "a", "list"]

Lists are indexed. This means that each position in the list has a number associated with it. When talking about a list position (or almost any position within coding), the index starts at 0.

For example:


    ["value1", "value2", "value3"]
        0	  1	     2

To get an item from a certain spot in a list, you use square brackets to define the index you want:


    a_list = ["this", "is", "a", "list"]
    
    print(a_list[0])
    # Result: "this"
    
    print(a_list[2])
    # Result: "is"

Additionally, you can get the length of a list:


    a_list = ["this", "is", "a", "list"]
    
    print(len(a_list))
    # Result: 4

This idea can be used for several different applications. One of the most useful, and most simple, is getting the last item from a list:


    a_list = ["this", "is", "a", "list"]
    
    last_item = a_list[len(a_list)-1]
    print(last_item)
    # Result: "list"

Remember! A list index starts at 0, so if you want to get the last item, you need to make sure that you subtract 1 from the length (4 - 1 = 3)

For Loops

A For loop is used for iterating over multiple items. For loops are often used with Lists, and are extremely helpful.

In python, For loops are really helpful for doing something multiple times, but to different items. For example, if you wanted to calculate the areas of several squares, you could use a For loop with a list of side lengths:


    sides = [2, 4, 7, 9]
    
    for length in sides:
        print('The area is '+ length * length)
    
    # Result:
    # The area is 4
    # The area is 16
    # The area is 49
    # The area is 81

This For loop is going through each item in the sides list, and assigning it to the variable length. Then, it is using that variable and calculating the area. The code above is the same as:


    sides = [2, 4, 7, 9]
    
    print('The area is ' + sides[0] * sides[0])
    print('The area is ' + sides[1] * sides[1])
    print('The area is ' + sides[2] * sides[2])
    print('The area is ' + sides[3] * sides[3])

Using a For loop allows for better performing code (as well as easier to read).

However! For loops don't need to only be used with lists! If you wanted to loop through each letter in a word:


    for x in "banana":
        print(x)

If you want to loop through a range of numbers:


    for x in range(2, 6):
        print(x)

For loops can get a bit more complicated. If you want to stop a loop, you can use a break statement. This could be helpful if you are looking for a certain value, and don't want to continue once you find it:


    for x in "abcdefghijklmnopqrstuvwxyz":
        if x == "e":
            print("I found the e!")
            break
    
    # Result: 
    # a
    # b
    # c
    # d
    # e
    # I found the e!

If you want to skip over a value, or simply not do anything, you can use pass:


    for x in ["dogs", "cats"]:
        if x == "dogs":
            pass
        else:
            print("hello")
    
    # Result:
    # hello

Lambda

Lambda is a small anonymous function. It can take any number of arguments, but can only have on expression.

That's confusing, what does it mean?

In simple terms, lambda defines one small and simple calculation that you want to complete within one line. Often, lambda expressions are used when filtering and sorting data. They take in a variable, and output the result of a calculation.

The focus is on Python right now, but I think this will help a bit with understanding. In Javascript, a lambda expression makes it slightly more obvious as to what is actually happening:


    let x = (x => x * 2) // lambda expression is within the parenthesis
    console.log(x(3))
    
    // Result: 6

Don't mind how the Javascript is written, but focus more on how the lambda expression is written. I personally like lambda expressions in JS because it is clear that you are taking a variable and doing something with it (for me, it's the arrow that helps show this):


    x => x * 2
    Take x as the variable, and then multiply it by two

In my opinion, you don't need to even know the word lambda to begin to understand what might be happening.

Now, back to Python. A lambda expression in python is written as lambda x : x * 2. This is the same as the Javascript above.

I'm going to include some code from W3 schools that explains some uses of a lambda expression. To be honest, I understand the idea, but not completely. It's a bit confusing, but I am going to do my best to explain after:

A function that doubles the number you give it:


    def myfunc(n):  
        return  lambda a : a * n  
        
    mydoubler = myfunc(2)  
        
    print(mydoubler(11))

A function that triples the number you give it:


    def myfunc(n):  
        return  lambda a : a * n  
        
    mytripler = myfunc(3)  
        
    print(mytripler(11))

Use a lambda function to create both of those methods in the same program:


    def myfunc(n):  
        return  lambda a : a * n  
        
    mydoubler = myfunc(2)  
    mytripler = myfunc(3)  
        
    print(mydoubler(11))  
    print(mytripler(11))

I personally would not worry too much about the W3 examples. They can get confusing, and I can't even wrap my head around how they take the arguments.

The most important idea to remember with lambda expressions, is that they are simple, generic expressions that can be used to simplify code and improve efficiency. Lambda expressions will reappear once I go over Maps.

Maps

The Map() function executes a specified function for each item in an iterable. It works similar to a For() loop, but is more compact and often easier to use once you understand how it works.

Take the For() loop below, for example:


    numbers = [1, 5, 2, 7, 4, 12, 16, 25, 19]
    larger_numbers = []
    
    for x in numbers:
        if x > 8:
            larger_numbers.append(x)
        else:
            larger_numbers.append(False)
    
    print(larger_numbers)
    # Result: [False, False, False, False, False, 12, 16, 25, 19]

This loop just takes numbers that are larger than 8 and creates a new list out of them, and then sets anything less than 8 as false. However, this code takes up 7 lines (not including extra lines or the print function). Using a Map(), this can be cut down to only 2:


    numbers = [1, 5, 2, 7, 4, 12, 16, 25, 19]
    
    larger_numbers = map(lambda x: x if x > 8 else False, numbers)
    
    print(list(larger_numbers))
    # Result: [False, False, False, False, False, 12, 16, 25, 19]

By using the Map function, we were able to shorten our code down to only two lines. While this is a very simple example, there are more complicated uses. Note the use of the list() function when printing the results.

For example, assume you have a list of coordinates in the format [[x,y], [x,y], ...]. You want only the value, if the x value is equal to a certain number. If the x is not equal to that number, you want it to set the y value as False, so that you can later perform other functions on it.


    coords = [[1,2],[3,4],[3,6],[3,9],[2,5],[7,3]]

In order to take only the y value from each coordinate, we can use a Map function:


    y_value = list(map(lambda x: x[1] if x[0] == 3 else False, coords))

Note that in this case, I needed to use an index number to get the proper coordinates. If you remember, a lambda function iterates through each value in a list. In this case, the main list is defined by the outer brackets of coords, with each separate item being the pair of coordinates. Because of this, we need to make sure to use the proper index to get either the x or y value from the pair of cords.

Putting this all together, we get:


    coords = [[1,2],[3,4],[3,6],[3,9],[2,5],[7,3]]
    y_value = list(map(lambda x: x[1] if x[0] == 3 else False, coords))
    
    print(y_value)
    # Result: [False, 4, 6, 9, False, False]

As you can see, we now have all the y values for when x is 3.

Now I mentioned earlier to pay attention to the fact that list() was used. This is because when you use the Map() function, it returns a Map object. If you were to print a map, you would get something along the lines of <map object at 0x55a487727be0>. Understanding what this really means is not super important right now, but it is important to remember that you must convert your map into a list before you try to print the result. The list() function does just this.

Lastly, you do not need to use a lambda function within a Map. The first argument within a Map function simply defines what you would like done to your list. If you wanted to perform something more complicated, you could pass in a different function.

The example below demonstrates how you could use this:


    def myfunc(n):  
        return  len(n)  
        
    x = map(myfunc, ['apple', 'banana', 'cherry'])
    print(list(x))
    # Result: [5, 6, 6]

# Example file showing a circle moving on screen
import pygame

# pygame setup
pygame.init()
screen = pygame.display.set_mode((1280, 720))
clock = pygame.time.Clock()
running = True
dt = 0

player_pos = pygame.Vector2(screen.get_width() / 2, screen.get_height() / 2)

while running:
    # poll for events
    # pygame.QUIT event means the user clicked X to close your window
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    # fill the screen with a color to wipe away anything from last frame
    screen.fill("purple")

    pygame.draw.circle(screen, "red", player_pos, 40)

    keys = pygame.key.get_pressed()
    if keys[pygame.K_w]:
        player_pos.y -= 300 * dt
    if keys[pygame.K_s]:
        player_pos.y += 300 * dt
    if keys[pygame.K_a]:
        player_pos.x -= 300 * dt
    if keys[pygame.K_d]:
        player_pos.x += 300 * dt

    # flip() the display to put your work on screen
    pygame.display.flip()

    # limits FPS to 60
    # dt is delta time in seconds since last frame, used for framerate-
    # independent physics.
    dt = clock.tick(60) / 1000

pygame.quit()

Example 2: Coin Game

The example below builds upon the given starter code to create a simple game. Use WASD to control your character, and collect 10 coins as fast as you can. This game uses features such as player movement, score keeping, a timer, and collision detection. 

import pygame
import random
import math

clock = None
running = None
dt = None

rand_coin_y = None

end = None

def restart():
    global clock, running, dt, collide, score, player_pos, rand_coin_x, rand_coin_y, start_ticks, seconds, end
    clock = pygame.time.Clock()
    dt = 0
    collide = False

    score = 0

    player_pos = pygame.Vector2(screen.get_width() / 2, screen.get_height() / 2)

    rand_coin_x = random.randrange(10,(screen.get_width()) -10)
    rand_coin_y = random.randrange(10,(screen.get_height()) -10)

    start_ticks=pygame.time.get_ticks()
    seconds = 0
    running = True
    
def run_game():
    global clock, running, dt, collide, score, player_pos, rand_coin_x, rand_coin_y, start_ticks, seconds, end
    while running:
        end = False
        seconds = round((pygame.time.get_ticks()-start_ticks)/1000, 2)
        keys = pygame.key.get_pressed()

            rand_coin_y = random.randrange(10,(screen.get_height()) -10)

Example 3: Platforming Test

The example below could work as starter code for a platforming game. This code has features such as player movement, collision detection, and jumping.

import pygame

pygame.init()
screen = pygame.display.set_mode((1280, 720))

clock = pygame.time.Clock()
running = True

player_rect = pygame.Rect(screen.get_width()/2, (screen.get_height()*.9)-100,20,20)
ground_rect = pygame.Rect(0, screen.get_height()*.90,screen.get_width(),screen.get_height()*.10)

gravity = 4

dt = 0

# Stores if player is jumping or not
isjump = False
   
# Force (v) up and mass m.
v = 7
m = 1

jump_delay = 0
seconds = 0

player_pos = pygame.Vector2(screen.get_width()/2, (screen.get_height()*.9)-100)

def run_game():
    global running, player_rect, ground_rect, gravity, dt, player_pos, isjump, v, m, seconds, jump_delay 
    while running:
        keys = pygame.key.get_pressed()
        
        for event in pygame.event.get():
            if event.type == pygame.QUIT or (event.type == pygame.KEYDOWN and event.key == pygame.K_ESCAPE):
                pygame.quit()
                
        screen.fill("white")
        
        collide = pygame.Rect.colliderect(ground_rect, player_rect)
        
        if not collide:
            player_pos.y += gravity
        
        if collide:
            gravity = 0
            player_rect.bottom = ground_rect.top
            player_pos.y = player_rect.bottom-10
        
        player_rect = pygame.Rect(player_pos.x-10, player_pos.y-10, 20, 20)
        
        pygame.draw.rect(screen, "blue", player_rect)
        pygame.draw.rect(screen,"green",ground_rect)
                
        if keys[pygame.K_a]:
            player_pos.x -= 300 * dt
        if keys[pygame.K_d]:
            player_pos.x += 300 * dt
            
        if isjump == False:
            seconds = (pygame.time.get_ticks()-jump_delay)
            
            if seconds >= 400:
            # if space bar is pressed
                if keys[pygame.K_SPACE]:
                    jump_delay = pygame.time.get_ticks()
                    # make isjump equal to True
                    isjump = True
                
        if isjump :
            # calculate force (F). F = 1 / 2 * mass * velocity ^ 2.
            F =(1 / 2)*m*(v**2)
            
            # change in the y co-ordinate
            player_pos.y -= F
            
            # decreasing velocity while going up and become negative while coming down
            v = v-1
            
            # object reached its maximum height
            if v<0:
                
                # negative sign is added to counter negative velocity
                m =-1
    
            # objected reaches its original state
            if v == -8:
    
                # making isjump equal to false 
                isjump = False
    
        
                # setting original values to v and m
                v = 7
                m = 1
                    
        # flip() the display to put your work on screen
        pygame.display.flip()

        # limits FPS to 60
        # dt is delta time in seconds since last frame, used for framerate-
        # independent physics.
        dt = clock.tick(60) / 1000
        
run_game()
